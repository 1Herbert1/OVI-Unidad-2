The biggest change that is currently planned for CSS level 3 is the introduction of modules. The advantage of modules is that it (supposedly) allows the specification to be completed and approved more quickly because segments are completed and approved in chunks. This also allows browser and user-agent manufacturers to support sections of the specification but keep their code bloat to a minimum by only supporting those modules that make sense.(lifewire, 2018, recoverd 21-11-1018, https://www.lifewire.com/what-is-css3-3466973)
